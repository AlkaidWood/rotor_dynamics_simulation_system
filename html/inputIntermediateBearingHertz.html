inputIntermediateBearingHertz

Input parameters about intermediate bearing considering Hertzian contact

Contents

  • Syntax
  • Description

Syntax

OutputParameter = inputIntermediateBearingHertz(InputParameter)

Description

InputParameter is a struct data saving all initial parameters except intermediate bearing parameters considering Hertzian contact.

OutputParameter is a strct data recording all InputParameter and the data about intermediate bearing considering Hertzian contact.

All of parameters about intermediate bearing considering Hertzian contact should be typed into this .m file.

All vectors in this .m file should be in column
